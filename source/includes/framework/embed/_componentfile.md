#### Files and File Component

##### Attributes

> Example JSON

```json
"attributes": {
  "isAutoUpload": "true"
}
```

The following attributes can be added to the Files and File Component to further customize the user experience:

Key | Value | Description
--- | ----- | -----------
isAutoUpload | true or false | Indicates if the component should auto-upload files when dropped onto the component but the running user(s).

<aside class="notice">
<b>Files</b><br/>
For the Files Component, you should also check out the Table Component attributes as the Files component supports the same features.
</aside>
